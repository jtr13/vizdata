Chapter 1 R Packages

1.1 Exploring data in packages:

Install datacat:

remotes::install("jtr13/datacat")

View information with:

library(datacat)
get_dataset_info("ggplot2") %>% View()

1.2 List of packages with data

1.2.1 alr4

Applied Linear Regression, 4th ed.

Datasets to Accompany S. Weisberg, Applied Linear Regression, 4th edition (2014). Many data files in this package are included in the alr3 package as well, so only one of them should be used.

library(datacat)
get_dataset_info("alr4") %>% DT::datatable()

1.2.2 dslabs

Data Science Labs

Rafael A. Irizarry, Amy Gill

Datasets and functions that can be used for data analysis practice, homework and projects in data science courses and workshops. 26 datasets are available for case studies in data visualization, statistical inference, modeling, linear regression, data wrangling and machine learning.

get_dataset_info("dslabs") %>% DT::datatable()

1.2.3 MASS

Modern Appplied Statistics with S

Functions and datasets to support Venables and Ripley, Modern Applied Statistics with S (4th edition, 2002).

get_dataset_info("MASS") %>% DT::datatable()
